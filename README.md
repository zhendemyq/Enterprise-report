# Enterprise-report

企业报表管理系统

## 项目结构

- `backend/` - 后端服务 (Spring Boot)
- `frontend/` - 前端应用 (Vue.js + Vite)

## 技术栈

### 后端
- Spring Boot
- MyBatis Plus
- Redis
- Sa-Token (认证授权)
- XXL-Job (定时任务)

### 前端
- Vue 3
- Vite
- Element Plus
- Vue Router
- Pinia

## 快速开始

### 后端启动
```bash
cd backend
./mvnw spring-boot:run
```

### 前端启动
```bash
cd frontend
npm install
npm run dev
```
